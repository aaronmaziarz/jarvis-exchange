# Jarvis Exchange

File exchange repository for sharing project files between Aaron and Jarvis.

## Purpose

This repository serves as a bridge for exchanging project files, assets, and deliverables. Use this for:

- Sharing code snippets and scripts
- Exchanging project documentation
- Passing generated files between sessions
- Collaborative project work

## Structure

- `inbox/` - Files for Jarvis to process
- `outbox/` - Files Jarvis has generated
- `projects/` - Ongoing project files
- `archive/` - Completed/historical files

## Usage

1. Place files in `inbox/` for Jarvis to process
2. Jarvis will move processed files to `outbox/` or `projects/`
3. Archive completed work to `archive/`
